# How to run the Code Examples in Codespace

## C
- gcc ./main.c -o ./main.app
- ./main.app

## C++
- g++ ./main.cpp -o ./main.app
- ./main.app

## C#
- mcs -out:./main.exe ./main.cs
- mono ./main.exe

## Go
- go run ./main.go

## Java
- javac Main.java
- java Main

## JavaScript
- node ./main.js

## Python
- python ./main.py
